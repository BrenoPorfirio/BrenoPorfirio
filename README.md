### 🚀 Sobre Mim

Sou um desenvolvedor que acredita que código de qualidade deve gerar **impacto direto no negócio**. Minha abordagem une a solidez técnica da computação com estratégias de crescimento e otimização:

* **Foco em Performance:** Desenvolvimento de aplicações rápidas e leves, priorizando métricas de Core Web Vitals.
* **CRO & Conversão:** Implementação de interfaces otimizadas para guiar o usuário e aumentar taxas de conversão.
* **SEO Técnico:** Estruturação de código e metadados para máxima visibilidade em mecanismos de busca.
* **UX (User Experience):** Design centrado no usuário para garantir navegabilidade intuitiva e acessibilidade.
* **Visão de Negócio:** Soluções pensadas para escalar e trazer retorno sobre o investimento (ROI).
